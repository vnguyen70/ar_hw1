This code uses takes an equirectangular projection panoramic image to create an Argon3 panorama. The input image should have a 2-1 aspect ratio and should not be greater than 4000x2000 pixels. This image is texture mapped onto the surface of a sphere surround the user/camera. This code can be expanded to include other images and functionality for a fuller Argon3 experience. Go to the site: http://ael.gatech.edu/argon for more details.
